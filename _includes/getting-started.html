Getting started with openFDA

1

Get your API key

We require API keys above a certain number of requests to manage load on the system, promote equitable access, and prevent abuse. See more about how to use your API key.

Signing up for an API key means you agree to our terms of service.

2

Learn how to search

OpenFDA has a special query syntax that may be different from other APIs you've used. To search for individual records:

search=field:term

To count records matching the unique values of a field, combine your search with count:

&count=field.exact

The API has powerful features, including timeseries generation. See more about openFDA query syntax.

3

Meet the data

Use this page's interactive queries to learn the endpoint, and take advantage of the field-by-field reference.
