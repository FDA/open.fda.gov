Welcome to openFDA!

There are a few things we need to make sure you're aware of before you begin experimenting with the API and associated data.

This API is not for clinical or production use. While we make every effort to ensure that data is accurate, you should assume all results are unvalidated. The data within the openFDA API is Public Domain and released under Creative Commons CC0.
This API is in beta. Do not assume it will stay online. As we continue work, we may need to change things around, rework queries, or move to a new URL.

You will see this message once a week throughout the beta period.
